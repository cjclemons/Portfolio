Here are some of my

  • Summarist

    Next.js, JavaScript, TypeScript, Redux, Firebase

    Summarist is a multi-page book discovery application where users can log in as guests or authenticate through Firebase. While both types of users can browse recommended and selected books with full access to text and audiobook summaries, only authenticated users can view premium content. The app also offers monthly and yearly subscription plans for users who want full premium access.

  • Ultraverse

    React, Git, Javascript, Node.js

    A dynamic NFT application where users can browse artwork and view real-time pricing along with a countdown timer for each piece. Built using Node.js, Axios, and a React-Slick carousel for smooth navigation, this project also reinforced my ability to manage version control through branching and pull requests on GitHub.

  • Library

    React, HTML, CSS, JavaScript

    Library is a multi-page e-commerce website that allows users to explore a wide range of books through dynamic routing, sort them using various filtering options, and add items to a fully functional shopping cart.

  • RMR

    React, HTML, CSS, JavaScript

    My first personal project using React, this movie rental app features a built-in search engine powered by an integrated API and multiple options for sorting movie results. Dynamic routing allows users to view detailed information for each selected film—including its summary, genre, runtime, and release year—along with a recommended movie section based on similar genres.

  • Treact Clone

    HTML, CSS, JavaScript

    This project is my first attempt at recreating a website's homepage, matching its responsive layout and key components while adding my own custom animations to elevate the user experience.
